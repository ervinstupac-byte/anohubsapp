THE "ONE-VANE" CATASTROPHE

If just ONE guide vane is misaligned (e.g. Vane #4 is at 25° while others are at 20°), it acts as a "flow disturber", creating a massive turbulent vortex.

This vortex hits the runner blades cyclically, leading to localized cavitation and shaft fatigue.

SYNCHRONIZATION SPECS

Parameter Specification Method
Sync Tolerance ± 0.5 mm Opening difference between vanes
Vertical Sealing 0.00 mm (Light-Tight) Feelers / Light Test at 0%

THE LIGHT TEST

Execute at 0% closure. Darkness required in runner chamber; high-power LEDs in spiral case.

If light is visible as a beam, the gap is > 0.2mm. ADJUSTMENT REQUIRED.

SHEAR PIN CHECK

If a shear pin breaks, the vane becomes "free-floating". It will flutter violently.

ACTION: Immediate emergency shutdown. A floating vane will destroy the runner.
